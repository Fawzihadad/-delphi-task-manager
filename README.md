# Delphi Task Manager

A simple console-based task manager built using Object Pascal (Delphi-style syntax).

## Features
- Add tasks
- View task list
- Mark tasks as completed
- Input validation

## Concepts Used
- Records (custom data structures)
- Arrays for storage
- Procedures for modular design
- Basic control flow (loops, conditionals)

## Future Improvements
- Dynamic data structures (lists)
- File persistence
- GUI version using VCL
